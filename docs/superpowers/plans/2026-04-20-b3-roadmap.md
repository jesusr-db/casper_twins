# B3 — Absorb caspers-kitchens Event Datagen Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate silent-503 deploys by shipping a self-healing deploy script (Phase 0), then absorb caspers' event datagen into the twins DAB so twins owns the upstream event stream (Phase 1).

**Architecture:** Two independent releases. Release 1 adds `scripts/deploy.sh`, a shell orchestrator over `bundle deploy → bundle run setup-lakebase → apps deploy`. Release 2 copies `raw_data.ipynb` + `canonical_data.ipynb` from caspers into `datagen/`, wires them as notebook tasks inside the existing `setup-lakebase` job, adds a DAB-managed `datagen-advance-time` scheduled job (landed PAUSED), declares the four schemas as DAB resources, and updates the destroy job to drop twins-owned base tables and pause the scheduler. Every artifact reaches production via `databricks bundle deploy` or a setup/destroy job task. No orphan scripts, no manual UI pokes.

**Tech Stack:** Databricks Asset Bundles (DAB), Databricks CLI, Databricks Python SDK, Unity Catalog, Jupyter notebooks (.ipynb), bash.

**Spec:** `docs/superpowers/specs/2026-04-20-b3-roadmap-design.md`

- [ ] **Step 3: Add a `_pause_datagen_schedule` helper function**

Add below `_safe`:

```python
def _pause_datagen_schedule(w: WorkspaceClient) -> None:
    """Pause twins-datagen-advance-time so it does not write while we destroy."""
    jobs = [j for j in w.jobs.list(name=DATAGEN_SCHEDULE_JOB_NAME)]
    if not jobs:
        log.info("  Not found (skip): %s", DATAGEN_SCHEDULE_JOB_NAME)
        return
    for job in jobs:
        try:
            w.jobs.update(
                job_id=job.job_id,
                new_settings={"schedule": {
                    "quartz_cron_expression": "0 */3 * * * ?",
                    "timezone_id": "UTC",
                    "pause_status": "PAUSED",
                }},
            )
            log.info("  Paused: %s (job_id=%s)", DATAGEN_SCHEDULE_JOB_NAME, job.job_id)
        except Exception as e:
            log.warning("  Failed to pause %s: %s", DATAGEN_SCHEDULE_JOB_NAME, e)
```

(If the SDK's `jobs.update` requires a different signature for schedule-only changes, adjust per the SDK reference — but this is the expected shape.)
